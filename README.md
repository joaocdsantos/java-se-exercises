# Java Standard Edition - Exercises

---

* Java version:  **17.0.2 2022-01-18 LTS**
* Developed during a **Java SE course**.

---

## Purpose of this repository

This repository was created to consolidate and reinforce core Java Standard Edition concepts through hands-on exercises.

Its goals are to:
- Refresh academic Java concepts
- Strengthen object-oriented thinking
- Practice clean code structure and class organization
- Prepare for more advanced Java frameworks (e.g., Spring Boot)
- Serve as a personal reference for future study

---

## Topics & Structure

Exercises are organized by topic under:
***src/main/java/exercises/***

### Packages

    1 - foundations             # Fundamentals: variables, operators, conditionals
    2 - loops                   # Loop exercises: for, while, do-while
    3 - vectors                 # Array (1D) exercises
    4 - matrix                  # Matrix (2D arrays) exercises
    5 - extra_exercises         # Additional logic and general Java practice exercises
    6 - oop1                    # OOP basics: classes and objects
    7 - oop2                    # OOP: inheritance exercises
    8 - oop3                    # OOP: polymorphism + advanced OOP
    9 - exceptions              # Exception handling: try/catch, custom exceptions
    10 - generics               # Generics: type parameters, generic classes and methods

---

### How to Run

1. Clone the repository
2. Open it in **IntelliJ IDEA**
3. Navigate to any exercise folder
4. Run the corresponding java file

Each topic contains independent exercises and can be executed individually.
